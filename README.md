# Student Task Manager Application

## Project Overview

This project was developed as part of the Professional Development Portfolio module.

The application allows users to manage tasks using a simple menu-driven interface built with Python.

## Features

* Add Tasks
* View Tasks
* Delete Tasks
* Save Tasks
* Load Tasks
* Error Handling

## Technologies Used

* Python 3
* JSON File Storage
* Visual Studio Code
* GitHub

## Testing

The application was tested for:

* Task Creation
* Task Viewing
* Task Deletion
* Data Persistence
* Error Handling
